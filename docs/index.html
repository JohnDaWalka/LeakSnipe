LeakSnipe Beta delivers instant seat HUD overlays, Monte Carlo equity calculations, automated hand history sync to Cloudflare R2/KV, and grounded LLM coaching powered by Grok and Ollama.

View GitHub Repository Explore Documentation (README.md)

Core Architecture Features

A decoupled hybrid desktop sidecar and Cloudflare edge pipeline built for ultra-low latency.

♠

Real-Time Live HUD Badge

Super-fast desktop seat overlay built with Tauri and Python window hooks. Displays live VPIP, PFR, 3Bet%, and postflop statistics tailored for CoinPoker, ACR, Bovada, and GGPoker.

⚡

Cloudflare MCP Server

Deployed Cloudflare Worker (`worker.leaksnipe.win/mcp`) featuring remote hand indexing, R2 bucket storage, KV `HAND_META` fast queries, and D1 database analytics.

🤖

Grounded AI Coaching

Automated leak detection powered by LLM models with direct JSON-RPC tools for hand retrieval, range charts, and deterministic equity calculations.

    PokerSite->>Importer: Writes Hand History File (.txt / .json)
    Importer->>DB: Parses and Stores Hand & Actions
    Sync->>DB: Queries Unsynced Hands (Pending Sync)
    Sync->>Worker: POST /mcp (store_large_hand_history)
    Worker->>R2: Saves Full Hand History JSON Object
    Worker->>KV: Stores Metadata Index (meta:handId)
    Sync->>DB: Updates _cloudflare_sync Table Status
